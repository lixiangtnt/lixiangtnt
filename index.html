智能制造的运用

智能制造作为下一代的制造技术，将深度的与互联网， 物联网，传感器，非接触式检测技术，大数据， 优化算法， 数据挖掘与预测相结合。 能够对市场需求提高快速的反应和决策，能够对客户定制的化的需求有所体现， 能够对全产业链以及对产业的全生命周期有一个有效的监控。

航空业一般都是先进制造技术，先进材料， 先进管理技术和先进商业模式的践行者。劳斯莱斯的航空发动机以租赁代替购买，使得对发动机的制造，维修，保养和再制造有了一个全方面的升级和提高。

国内的智能制造的认识似乎有一定的误区和局限性， 一味的强调互联网化自动化，事实上机器替代人只是上一代高度自动化的衍生和发展，并不是新一代技术的智能化全方位革命的演进。

以下文章是匆匆花五分钟找到的一些工业自动化的框图性概述。更为全面和深刻的综述期待有时间完成后和大家分享。

Custom Domain

Posted on 2016-11-19

custom domain setting in git bash

Add your domain into hexo

Modify the url in _config.yml to look like the following. Replace mydomain.com with your own domain and remember to save the file. [ref]

URL

url: http://mydomain.com
root: /

Deployment

Configure git deployment by adding following into _config.yml and saving the file. Replace username with your github username.
Docs
deploy:
type: git
repo: https://github.com/lixiangtnt/lixiangtnt.github.io (Please replace lixiangtnt with your own domain page, Not form like git@github.com:jr0cket/jr0cket.github.io.git)
branch: master

Creating CNAME in file

Github uses file called CNAME to redirect your site. If you read the official guide it says the file must saved under the root directory in the master branch. But remember that for us the source branch is where the raw material is and master branch is generated when you deploy. So if you put something in the master branch it will be overwritten next time you deploy your site.

Create file called CNAME (all caps, no extension) under the source directory in the source branch and write your domain name into that file. Replace mydomain.com with your own domain.

$ cd source/
$ echo mydomain.com > CNAME
